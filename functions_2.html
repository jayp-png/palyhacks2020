In this tutorial you'll learn about functions' parameters in JavaScript.

Explanation

JS's arrays are one of the reasons the language is so powerful. A array is a special type of variable that can store many values in it. Instead of creating dozens of variables manually, you can use an array.

Creation

There are several ways to create an array. The easiest is to just assign it to a variable.

let languages = ["C", "Python", "JavaScript", "PHP"];

This is called an array literal.

If you want to create an empty array with a large number of elements, you can do this:

let bigArray = new Array(500);

Access

This is the syntax to access a specific element of an array:

array[elementNumber]

For instance, if you wanted to get the 3rd item in the languages array, you would do array[2].

Wait. Why 2, not 3? Is that a typo? you might be asking. Actually, in most languages, arrays start counting from zero.
