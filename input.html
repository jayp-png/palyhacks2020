Alert and Prompt

In this tutorial you'll learn about alerts and prompts in JavaScript.

Explanation

JavaScript has two built-in methods for showing information to the user, "alert" and "prompt".

Alert

To show a small popup with text to the user, you can use "alert(text)".

Prompt

"prompt" is similar to "alert", except "prompt" can return what the user typed.

let input = prompt("Enter some text.");
console.log("You typed " + input);
