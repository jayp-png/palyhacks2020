Arrays

In this tutorial you'll learn about arrays in JavaScript.

Explanation

The arrays of Javascript are one of the reasons this language is so powerful. An array is a special datatype that acts as a list. Instead of creating dozens of variables manually, you can store and use them as an array.

Creation

There are several ways to create an array, but the easiest way it to assign many values to 1 variable.

let languages = ["C", "Python", "JavaScript", "PHP"];

This is called an array literal.

To create an empty array with a large number of elements, you can do this:

Over here you are creating an array named "bigArray" with 500 values stored inside of it.
let bigArray = new Array(500);

Access

This is how you access a specific value inside of an array:

array_name[Index]

For instance, if you wanted to get the 3rd item in the "languages" array, you would do languages[2].

You might be asking,"Wait. But why 2, not 3? Is that a typo?." Actually, in most languages, arrays start counting from zero.

So the first item of a list would have index 0, the second item of a list would have index 1, and so on.
