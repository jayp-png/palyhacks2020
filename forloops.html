Explanation

A big part of JS is variables. A variable is a piece of data with a name. You can change its value throughout the program.

Creation

You can create a variable with the let keyword. You can create a variable and assign something to it later, or immediately give it a value.

/* Assign at creation. */
let myVariable = 5;

/* Give value later. */
let otherVariable;

otherVariable = "Tacos";

Usage

To refer to the value of a variable, just use its name.

Use the = to assign a value to the variable.

let breakfast = "pancakes";
console.log(breakfast);     /* This prints "pancakes". */

breakfast = "waffles";
console.log(breakfast);     /* What does it print now? */

What can you store in variables?

Variables in JS can store a few types of data.

  • Numbers
  • Strings (text)
  • Booleans (true/false)
  • Lists
  • Objects

In this tutorial you learned about the basics of numbers and strings. In the future you'll learn about other datatypes.
