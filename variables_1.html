Variables 1

In this tutorial you'll learn about the basics of variables.

Explanation

A big part of JS is variables. A variable is a value assaigned a name.

You can change the value of a variable throughout the program.

Creation

You can create a variable with the "let" keyword.

A variable is assigned a name and value upon creation.

Don't worry, you can change its value later in the program.

/* Assign at creation. */
let myVariable = 5;

/* Give value later. */
let otherVariable;

otherVariable = "Tacos";

Usage

To refer to the value of a variable, just use its name.

Use the "=" to assign a value to the variable.

let breakfast = "pancakes";
console.log(breakfast);     /* This prints "pancakes". */

breakfast = "waffles";
console.log(breakfast);     /* What does it print now? */

What can you store in variables?

Variables in JS can store a few types of data.

  • Numbers
  • Strings (text)
  • Booleans (true/false)
  • Lists
  • Objects
